Are Facebook-likes a valuable input for the quality of a movie?

The dataset


The dataset used for this visualization is provided by kaggle.com and contains 5000 movies containing loads of data about the films from IMDB

The story

With this visualization, I want to show how a movie's Facebook-likes and its IMDB rating relate to each other. If you look at the graph you can see that usually, a higher number of Facebook-likes means the film is rated higher. However, this only goes one way, there are a lot of movies with a high rating but with zero to very little Facebook-likes.
Then there are of course some outliers, films with a high number of Facebook-likes but a low rating. Can you guess which film has 62 thousand likes but a rating of 1.6?
